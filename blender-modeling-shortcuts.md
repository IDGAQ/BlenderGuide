Object Mode
```
S: scale
R: rotate
G: grab/move
(S/R/G->X/Y/Z: move horizontally or vertically)
(shift+X/Y/Z: scale in other two directions)

N: toggle sidebar(transform, tool, view)

// highlight multiply objects
M: move to collection

Shift+A: add

Shift+Z: switch to wireframe mode

Shift+D: duplicate

Shift+S: cursor to selection/ selection to cursor

Shift+F: fly
(+Q/E: fly up or down)

shift+tab: make grab snapped, move more pixels at a time

// make current transformation 0 instead of initial global rotation
ctrl+A: applyF

// set origin
Object-> Set Origin

// goto/find/focus on selected object
click the object -> view -> frame selected

// select multiply objects and the parent object(which u want the data from)
ctrl+L: link data
```



Edit Mode
```
Tab: switch mode(object<->edit)
1: vertex select
2: edge select
3: face select

ctrl+R: LOOP CUT

Z: views UI
ctrl+L: link UI

alt + left click: loop select

//分手指，分脚趾
alt + E: extrude (face)

E: extrude

M: merge(all vertex)

/// select face:
P: separate to a new mesh
K: cut, use knife to add new edges

// select multiple vertices(add new edges between vertices):
J: join

/// select vertice
ctrl + E: edge OPTIONS

i: insert face
i -> b: insert face without boundary

// Face select
ctrl + num+ :select more
ctrl + num- : select less


/// edge select
ctrl + X: remove edge

f: fill/add faces

// select all vertices/faces/edges
a: select all
// 添加斜面
ctrl+b: bevel

F3: Find all shortcut reference

shift+ctrl+alt+s: shear
```
