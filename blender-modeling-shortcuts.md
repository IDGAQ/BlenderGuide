Object Mode
```
S: scale
R: rotate
G: grab/move
(S/R/G->X/Y/Z: move horizontally or vertically)
(shift+X/Y/Z: scale in other two directions)

N: toggle sidebar(transform, tool, view)

Shift+A: add

Shift+D: duplicate

Shift+S: snap cursor to...

Shift+F: fly
(+Q/E: fly up or down)

shift+tab: make grab snapped, move more pixels at a time

// make current transformation 0 instead of initial global rotation
ctrl+A: applyF

// set origin
Object-> Set Origin
```

Edit Mode
```
Tab: switch mode(object<->edit)
1: vertex select
2: edge select
3: face select

ctrl+R: LOOP CUT

Z: views UI
ctrl+L: link UI

alt + left click: loop select

//分手指，分脚趾
alt + E: extrude (face)

E: extrude

M: merge(all vertex)

ctrl + E: edge OPTIONS

i: insert face

// Face select
ctrl + num+ :select more
ctrl + num- : select less

f: fill
// 添加斜面
ctrl+b: bevel

F3: Find all shortcut reference

shift+ctrl+alt+s: shear
```
