Object Mode
```java
S: scale
R: rotate
G: grab/move
(S/R/G->X/Y/Z: move horizontally or vertically)

Shift+A: add

Shift+D: duplicate

Shift+F: fly
(+Q/E: fly up or down)
```

Edit Mode
```java
Tab: switch mode(object<->edit)
1: vertex select
2: edge select
3: face select

ctrl+R: LOOP CUT

Z: views UI
```
