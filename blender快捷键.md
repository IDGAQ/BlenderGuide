Object Mode
```
S: scale
R: rotate
G: grab/move
(S/R/G->X/Y/Z: move horizontally or vertically)

N: toggle sidebar(transform, tool, view)

Shift+A: add

Shift+D: duplicate

Shift+F: fly
(+Q/E: fly up or down)
```

Edit Mode
```
Tab: switch mode(object<->edit)
1: vertex select
2: edge select
3: face select

ctrl+R: LOOP CUT

Z: views UI
ctrl+L: link UI
```

Mirror Modifier:
```
1.find the wrench icon
2. add modifier(mirror)
```
